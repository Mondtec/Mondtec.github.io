Igual que una interfaz de usuario, permite la interacción y comunicación entre un software y una persona, una API viene del acrónimo de Application Programming Interface

Cada API está diseñada en un lenguaje de programación concreto y con unas especificaciones distintas que la definen (las APIs pueden incluir especificaciones para estructuras de datos, clases de objetos o variables, a partir de las cuales se basa el uso de esa interfaz).

Existe una gran variedad de API's que pueden ser consumidas por los desarrolladores con el fin de obtener datos. La práctica más común es realizarla por medio de un API RESTFUL. El gobierno federal de México, desde hace ya unos años, implementó una forma en la que los usuarios accedan a datos libres a través de dicho consumo de API's creadas por ellos.

Se pone a dispocisión el siguiente link donde se puede indagar todo lo que comparte el gobierno de México a través de los diversos API's que maneja: https://datos.gob.mx/desarrolladores
